Delicious Profile Cards

React.js | Express.js | SQLite | HTML5 | CSS3 (Flexbox, Grid) | SASS

Refactorización a React.js de un código heredado en Vanilla Javascript. Además se ha creado un servidor propio con Node.js, Express.js y SQLite.

Rick and Morty Character Finder

React.js | HTML5 | CSS3 (Flexbox, Grid) | SASS

Buscador de personajes de la serie Rick and Morty. Programada en React.js, los datos de los personajes se extraen de una API a través del mecanismo Fetch. Consta de tres filtros de búsqueda y, de cada personaje, se puede consultar información más detallada con la creación de una segunda tarjeta de datos. Dispone de un botón de retorno, en donde queda almacenada la búsqueda realizada.

Series Finder

Vanilla Javascript | HTML5 | CSS3 (Flexbox, Grid) | SASS

Buscador de series cuyos datos se extraen de una API a través del mecanismo Fetch. Programado en Vanilla Javascript, permite almacenar en el local storage las series seleccionadas como favoritas a través de un click sobre la carátula.

Look Like Loop Cards

Vanilla Javascript | HTML5 | CSS3 (Flexbox, Grid) | SASS | Gulp

Aplicación para generar tarjetas de visita virtuales programada con Vanilla Javascript. En ella el usuario introduce los datos requeridos, hace el envío de los mismos y el servidor le devuelve una URL con la tarjeta, que puede ser compartida en Twitter a través de un botón que se incluye en la web. Asimismo, los datos quedarán almacenados en el local storage para una mejor experiencia de usuario.

Pokemon

React.js | HTML5 | CSS3 (Flexbox, Grid) | SASS

Listado de personajes de Pokemon, cuyos datos se extraen de un fichero .json, y se renderizan usando como soporte React.js. La funcionalidad de esta web es la posibilidad de seleccionar cada tarjeta del personaje para señalarla como favorita.
